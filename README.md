# ResQNet

## Overview
ResQNet is a disaster management web platform designed to provide critical information and support during emergencies. The platform offers real-time alerts, interactive maps, shelter location services, and resource tracking.

## Features
- Real-time disaster alerts
- Interactive maps
- Shelter locator
- Resource tracking
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
